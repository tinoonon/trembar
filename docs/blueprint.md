# **App Name**: Trem Bar Sabores

## Core Features:

- Public Menu & Item Browsing: Users can view the full menu, including categories, dish descriptions, prices, and highlight badges, with filtering capabilities.
- Interactive Event Listing: Showcase upcoming events with dedicated pages, images, dates, descriptions, and options to learn more or reserve.
- Online Table Reservation System: A user-friendly form for customers to book tables, specify date, time, number of guests, and special occasions, with automated confirmation via email/WhatsApp.
- Restaurant Gallery & Ambiance Showcase: A dynamic masonry grid photo gallery displaying high-quality images of dishes, drinks, ambiance, and events, with a lightbox for detailed viewing.
- Admin Content Management (CRUD): A secure administrative panel enabling content managers to perform CRUD operations on menu items, events, manage uploaded gallery images, and view customer inquiries.
- Admin Reservation & Contact Management: An interface for administrators to efficiently review, update the status (confirm/cancel) of incoming reservations, and manage customer contact messages.
- AI-Powered Content Generation Tool: An integrated tool in the admin panel to assist with generating engaging and tone-appropriate descriptions for new menu items or promotional event announcements.

## Style Guidelines:

- Main background color: Preto Café (#1A0A00) to establish a deep, rustic-elegant atmosphere, reminiscent of classic coffee.
- Secondary color: Marrom Escuro (#3B1A08) for alternating sections and textural depth, complementing the primary background.
- Primary accent color: Marrom Sienna (#8B4513) to highlight interactive elements such as buttons and key calls to action.
- Decorative accent color: Dourado Envelhecido (#C8922A) used for titles, ornamental details, and subtle premium accents, echoing Art Déco aesthetics.
- High-contrast text color: Creme (#F5ECD7) ensuring optimal readability against the dark background and primary colors.
- Neutral base color: Branco Puro (#FFFFFF) reserved exclusively for the logo, essential icons, and areas requiring stark contrast.
- Display font: 'Playfair Display' (serif) for main titles, contributing an elegant and classic feel.
- Script font: 'Dancing Script' (script) for expressive taglines and signatures, adding a personal and artistic touch. Note: currently only Google Fonts are supported.
- Body font: 'Lato' (sans-serif) for clean and highly readable body text, ensuring content is easily digestible across all devices. Note: currently only Google Fonts are supported.
- The official 'Trem Bar' logo, featuring a spoon with steam, train rails, and serif typography, will be prominently integrated across the site.
- Icons will maintain a consistent aesthetic, potentially incorporating subtle gold outlines or fills in line with the Art Déco ornaments.
- The overall aesthetic is rustic-elegant, drawing inspiration from old Minas Gerais train stations, featuring dark wood and aged iron textures.
- Responsive layout designed mobile-first, ensuring optimal presentation and user experience on devices of all sizes.
- Subtle micro-animations such as fade-in effects for elements on scroll, gentle parallax for background images, and a sophisticated golden glow on hover for interactive elements.
